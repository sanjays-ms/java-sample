## Learning Paths

### Path 1: Java Basics (2-3 hours)
For those new to Java:

1. [Variables and Data Types](./documentation/01-variables.md)
2. Operators
3. Control Flow
4. Arrays
5. Strings
6. Methods

### Path 2: OOP Fundamentals (3-4 hours)
For understanding object-oriented programming:

1. Classes and Objects
2. Encapsulation
3. Inheritance
4. Polymorphism
5. Abstraction
6. Interfaces

### Path 3: Modern Java (2-3 hours)
For experienced developers learning Java 21+ features:

1. Records
2. Sealed Classes
3. Pattern Matching
4. Text Blocks
5. Virtual Threads
