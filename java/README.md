# Java Documentation and Cheatsheets

A complete Java learning resource targeting Java 21+ with detailed documentation and quick-reference cheatsheets.

## Getting Started

### New to Java?

Follow the **Recommended Learning Path** in order:

1. **[Variables and Data Types](./documentation/01-variables.md)** - Start here! Primitives, type casting
2. **[Operators](./documentation/02-operators.md)** - Arithmetic, logical, bitwise
3. **[Control Flow](./documentation/03-control-flow.md)** - if/else, switch, loops
4. **[Arrays](./documentation/04-arrays.md)** - Single-dimensional, Arrays utility class
5. **[Multi-Dimensional Arrays](./documentation/05-multidimensional-arrays.md)** - 2D arrays, jagged arrays
6. **[Strings](./documentation/06-strings.md)** - String, StringBuilder, text blocks
7. **Methods** - Parameters, overloading, varargs
8. **Classes and Objects** - Constructors, this, static
9. **Encapsulation** - Access modifiers
10. **Inheritance** - extends, super
11. **Polymorphism** - Runtime polymorphism
12. **Abstraction** - Abstract classes, interfaces
... (more topics coming)
